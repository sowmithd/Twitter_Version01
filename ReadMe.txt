This application has following features:
1.Registration
2.Login/Logout/Forgot Password
3.Profile display with pictures(from database)
4.Follow/Unfollow
5. Trends
6. Tweet
7. Comment
8. Retweet(share the existing tweet)
9. Messaging
10. Display tweets of the people whom user follows

Note: This application is based on database hosted on "mis-sql.uhcl.edu",
for using this code, database connection codes should be modified.
